A wrapper around the [Quotable.io API](https://api.quotable.io).

```go
import (
	"fmt"
	"github.com/UltiRequiem/quotable/pkg"
)


func main(){
        data, error := quotable.GetQuotable()

        fmt.Printf("%s\n -%s\n", data.Content, data.Author)
}
```
